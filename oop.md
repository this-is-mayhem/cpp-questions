# `mutable`
В C++ модификатор `mutable` разрешает изменять поле класса даже в `const` объекте. Обычно `const` объект запрещает изменять его члены, но `mutable` как бы снимает это ограничение.
## `mutable` в `const` методе
```
class Bar {
    mutable int cache = 0;

public:
    int getSomething() const {
        if (cache == 0) {
            cache = 42;  // ✅ можно менять, даже в const-методе
        }
        return cache;
    }
};
```
`getSomething()` объявлен как `const`, значит:
- он не может менять немутируемые поля объекта,
- он может менять только mu`t`able поля.

Таким образом, `mutable` позволяет реализовать логическую константность (logical constness), когда метод считается "константным снаружи" (ничего не меняет с точки зрения пользователя класса), но внутри может вести служебные изменения: кеши, счётчики, ленивые вычисления и т.п.

# Виртуальный деструктор и конструктор
## Деструктор
Реально существуют и очень важны при работе с полиморфизмом.

**Проблема:** если есть базовый класс и от него наследуется подкласс:
```
class Base {
public:
    ~Base() { std::cout << "Base destructor\n"; }
};

class Derived : public Base {
public:
    ~Derived() { std::cout << "Derived destructor\n"; }
};
```
И создается объект через указатель на базовый класс:
```
Base* obj = new Derived();
delete obj; // ⚠️
```
Будет вызван только деструктор `Base`. Деструктор `Derived` не вызовется, что приведёт к утечкам ресурсов, если в `Derived` есть динамическая память или другие ресурсы.

❗ ***Деструктор базового класса почти всегда должен быть виртуальным, если класс предполагается как базовый для полиморфных объектов.***

## Конструктор
Конструкторы не могут быть виртуальными. Полиморфизм работает через указатель на объект, а объект создаётся уже после вызова конструктора. На момент вызова конструктора объекта ещё нет `vtable`, поэтому виртуальность недоступна.

🔹 **Для имитации** обычно используют виртуальную функцию `clone()` или фабричный метод:
```
class Base {
public:
    virtual ~Base() {}
    virtual Base* clone() const = 0;
};

class Derived : public Base {
public:
    Base* clone() const override {
        return new Derived(*this);
    }
};

void example() {
    Base* b = new Derived();
    Base* copy = b->clone(); // создаём копию полиморфно
    delete b;
    delete copy;
}
```
